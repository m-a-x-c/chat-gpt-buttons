## Full description

ChatGPT keeps shuffling its menus. Web Search ends up under "More", then back at the top, then two clicks deep. Deep Research moves. Agent Mode moves. The model switcher gets a redesign every few weeks. Your muscle memory becomes a liability.

ChatGPT Shortcut Buttons pins the buttons. A row of one-click toggles sits right next to the chat box, mirroring ChatGPT's own state. Click "Web search" — Search turns on. Click "Deep research" — Deep Research turns on. Click the paperclip — the file picker opens. No menus, no hunting, no muscle memory rewrites.

**What gets shortcut**

- 📎 **Upload** — opens the file picker directly.
- 🧠 **Standard thinking** / **Extended thinking** — toggles model reasoning effort and switches model in one click.
- 🌐 **Web search** — turns on Search.
- 🖼 **Create image** — flips on image generation.
- 🔬 **Deep research** — enters Deep Research mode.
- 🤖 **Agent mode**, **Add sources**, **Canvas**, **Quizzes** — every other feature normally buried under "More".

**Live state sync.** Each button mirrors ChatGPT's own state. Toggle a feature on through ChatGPT's native UI and the matching button lights up; toggle a button and ChatGPT updates accordingly.

**Configurable.** Click the toolbar icon to choose which buttons to show. Defaults to the five most-used (Upload, Standard thinking, Extended thinking, Web search, Create image, Deep research). Settings sync across your devices via Chrome's built-in sync.

**Pause anytime.** A master toggle in the popup disables the extension entirely without uninstalling.

**No account, no tracking, no servers.** The extension only runs on `chatgpt.com`. It stores preferences locally (via `chrome.storage.sync`) and never transmits anything to a third party.

## Main features

- One-click pills for Standard thinking, Extended thinking, Web search, Create image, Deep research, Agent mode, Add sources, Canvas, Quizzes, and Upload.
- Live two-way state sync with ChatGPT's own pills and model switcher.
- Per-button visibility settings (popup with toggle switches).
- Master enable/disable switch.
- Survives ChatGPT's SPA navigation — no page refresh needed.
- Zero network access; zero data collection beyond local preferences.
